# H1101-CP2-GRP26

Project Description:
The project focuses on the development and refinement of the MotorPH application graphical user interface (GUI). The system is composed of several key modules represented by the project files: The process began with building the main application interface through **MotorPH_GUI.java**, providing navigation and access to system features. From there, **MotorPH_LoginValidation.java** handles user authentication and ensures that only authorized users can access the system. The **Employee_Form_UI.HTML** file contains the employee information form prototype used for collecting and displaying employee data, while **Add_Button_UI.java** implements functionality for adding new employee records within the application. Finally, the team applied **UI Polish** component to refine the visual design, improve overall consistency, and user experience of the interface ensuring all components were properly integrated and fully functional, working towards a cohesive and operational MotorPH desktop application.
