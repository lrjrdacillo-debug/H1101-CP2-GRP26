# H1101-CP2-GRP26

Project Description:
The system is composed of several key modules represented by the project files. The MotorPH_GUI.java file serves as the main application interface, providing navigation and access to system features. MotorPH_LoginValidation.java handles user authentication and ensures that only authorized users can access the system. The Employee_Form_UI.HTML file contains the employee information form prototype used for collecting and displaying employee data, while Add_Button_UI.java implements functionality for adding new employee records within the application. The UI Polish component focuses on improving the visual design, consistency, and overall user experience of the interface.
